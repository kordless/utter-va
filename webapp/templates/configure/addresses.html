Appliance Addresses

Addresses are Bitcoin Destination Addresses which are managed by a Coinbase Account tied to this Appliance. This appliance advertises payment addresses via Instance Asks on the Pool Operator site. Addresses will be recycled for use with new instances after an older instance is Decommissioned.

The Bitcoin Address links may be used to view payment transactions. When instances are attached to a Bitcoin address, you may click on the Instance ID to view instance details.
