Flavors & Ask Prices

About Flavors

Flavors are used by Appliances to describe Instance Type values for # of Compute Cores, Memory, Disk and Network Speed. You can update this appliance's Ask Prices for flavors it advertises via Instance Asks by clicking on the Flavor Name link.

Flavors are synced every 15 minutes from the Pool Operator and are automatically installed into the OpenStack Cluster by the appliance. You should only enable serving flavors your OpenStack cluster can readily support.
