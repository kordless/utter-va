Getting Started

Fill out the following form to configure your appliance. This appliance needs to have a Regsitered API Pool Token, a valid & unique Coinbase API Key and Secret and an Ngrok API Token before it will function. Toggle visibility of the Coinbase API Secret with the icon.

Note: You should ensure this appliance is using a unique Coinbase Client ID if you are running multiple appliances. Multiple IDs can be created in a single Coinbase account.

Signup for an Ngrok account by clicking on the icon. Use the Image Cache IP Address field to enter an accessible IP for this Virtual Appliance. Set the map to the approximate location of your OpenStack Cluster so users can request instances by Geolocation.
